# MSPM0G3507 Motion Control Firmware

## Project Summary

This repository contains the **motion control firmware** for the 2025 National Undergraduate Electronic Design Contest (NUEDC) Autonomous Aiming System.

The firmware is developed on the **Texas Instruments MSPM0G3507** platform using **DriverLib** and **SysConfig**. It is responsible for the low-level control of the mobile robot, including motor driving, encoder acquisition, autonomous line following, and communication with the STM32 master controller.

---

## Main Features

* PWM motor control (TB6612FNG)
* Quadrature encoder acquisition
* 8-channel grayscale sensor line following
* Differential steering control
* Multi-lap autonomous navigation
* UART communication with STM32
* GPIO interrupt processing
* Modular DriverLib-based software architecture

## Usage

1. Import the project into CCS.
2. Install the latest MSPM0 SDK.
3. Open the `.syscfg` file and regenerate DriverLib configuration if necessary.
4. Compile and download the firmware to the MSPM0G3507 development board.
5. Connect the motor driver, encoders, grayscale sensors, and STM32 according to the hardware design.

---

## Notes

* This firmware is designed to work together with the STM32 master controller.
* The STM32 is responsible for high-level motion planning and vision processing.
* The MSPM0G3507 focuses on real-time motor control and sensor acquisition.
* If pin assignments are modified, regenerate the configuration using SysConfig before rebuilding.
